Miniguia de Estudos em Python com NotebookLM

Projeto desenvolvido durante o desafio da DIO com o objetivo de construir uma base de conhecimento utilizando o Google NotebookLM para apoiar os estudos de Python voltados para Análise de Dados, Automação de Processos e Ciência de Dados.

Sobre o projeto

Este projeto utiliza o Google NotebookLM como uma ferramenta de apoio aos estudos.

A ideia é transformar materiais de estudo (PDFs, artigos, apostilas e vídeos) em uma base de conhecimento organizada, permitindo que a IA responda dúvidas utilizando apenas fontes previamente selecionadas.

Dessa forma, o NotebookLM funciona como um assistente especializado em Python, auxiliando na revisão de conteúdos, criação de resumos, explicação de conceitos e organização do aprendizado.

Objetivos

- Construir um guia de estudos de Python do nível iniciante ao avançado.
- Consolidar conteúdos de Programação Orientada a Objetos (POO).
- Organizar materiais sobre Python para Ciência de Dados.
- Reunir conteúdos sobre automação de processos.
- Criar uma base de conhecimento reutilizável para futuras consultas.
- Utilizar Engenharia de Prompts para obter respostas mais precisas da IA.
