# 📝 Fontes de Texto e Guias Teóricos

Este repositório consolida materiais de leitura, roadmaps de carreira e exercícios de fixação para dar suporte conceitual ao aprendizado de Python.

---

## 🗺️ Planejamento e Roadmaps de Carreira

### 1. [Roadmap Python: Como Estudar Python Passo a Passo](https://academify.com.br/roadmap-python/)
* **O que focar:** Use este guia para entender a ordem cronológica de aprendizado do ecossistema Python. Ele é perfeito para alinhar suas expectativas sobre o que estudar primeiro (sintaxe básica) antes de tentar avançar para frameworks complexos.
* **Uso prático:** Mapeamento de progresso e definição de marcos semanais.

### 2. [Comprehensive Python Learning Path: Beginner to Expert](https://www.coursera.org/resources/python-learning-roadmap)
* **O que focar:** Este guia da Coursera foca no desenvolvimento profissional. Ele ajuda a entender o mercado de trabalho, quais bibliotecas são mais exigidas e como conectar Python com Ciência de Dados, Engenharia e Desenvolvimento Web.
* **Uso prático:** Planejamento de carreira de longo prazo e identificação de tecnologias complementares.

---

## 🧱 Estruturas de Dados e Teoria Avançada

### 3. [Quando usar Listas, Tuplas, Conjuntos e Dicionários em Python](https://www.dio.me/articles/quando-usar-listas-tuplas-conjuntos-e-dicionarios-em-python-sem-decorar-entendendo-a-ideia-0e014b16e6ac)
* **O que focar:** A "regra de ouro" para a escolha de estruturas. Em vez de apenas decorar comandos, este artigo ensina o raciocínio lógico por trás da mutabilidade e performance de cada coleção.
* **Uso prático:** Consulta rápida na hora de decidir onde e como armazenar os dados do seu programa (ex: usar Tuplas para dados fixos ou Dicionários para buscas rápidas).

### 4. PDFs Complementares de Aulas de Python (IFB - Prof. Henrique)
* **O que focar:** Base acadêmica sólida. Estes PDFs são ideais para revisar conceitos de lógica computacional, fluxo de controle estruturado e algoritmos fundamentais.
* **Uso prático:** Revisão teórica rápida de sintaxe antes de partir para a digitação de códigos mais complexos.

---

## 🎯 Prática de Programação

### 5. [Python Exercises, Practice, Solution - w3resource](https://www.w3resource.com/python-exercises/)
* **O que focar:** Este é o coração da sua prática diária. O site oferece centenas de exercícios resolvidos e divididos estritamente por tópicos (como listas, strings, condicionais e POO).
* **Uso prático:** Todo dia, após estudar a teoria de um tópico, resolva de 3 a 5 exercícios desta base para fixar a sintaxe na memória de longo prazo.
